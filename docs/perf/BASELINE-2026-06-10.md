Google's hosted PageSpeed Insights API was rate-limited (HTTP 429, keyless quota), so these are local numbers; compare future runs against these using the SAME method on the SAME machine. Production commit at time of run: `dfe6262` content + docs merge `808123d`.

Reproduce: `npx -y lighthouse <url> --chrome-flags="--headless=new" --only-categories=performance,accessibility,best-practices,seo --output=json` (add `--preset=desktop` for desktop).
